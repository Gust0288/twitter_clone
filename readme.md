# Twitter Clone (IN PROGRESS)

## Project Overview

This is a Twitter clone application built as a web development project. It replicates core Twitter functionality.

## Features

- User registration and authentication
- Profile creation and customization
- Creating and deleting posts ("tweets")
- Following/unfollowing other users
- Timeline of posts from followed users
- Like and comment functionality
- User notifications
- Search functionality

## Tech Stack

- **Backend**: PHP
- **Database**: MySQL
- **Frontend**: HTML, CSS, JavaScript
- **Version Control**: Git
